# 使用说明

1. 请和[自定义配置](../../Input_Method/万象拼音/Rime4Hamster/)一起使用，尤其是自定义按键部分
2. 打开元书输入法键盘皮肤中的“启动皮肤开发者模式”，导入后点击皮肤，或者在导入皮肤后长按手动运行`main.jsonnet`，两种方式来生成皮肤配置文件
